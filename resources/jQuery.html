jQuery


jQuery is a fast phased, small library with additional features to JavaScript. The main advantages of using
jQuery is, that HTML document traversal and manipulation, event handling and animations are on ex. web pages
much simpler to implement in jQuery. jQuery is accessed via basic JavaScript.
The dollar sign, $, works as a selector for a given tag, id or class declaration etc. in HTML
so the jQuery language can manipulate with the given HTML-content.
