Arrays and loops

Example with discarded numbers - array, loop and if statement

The syntax of arrays and loops in JavaScript are quite similar to Java itself. The major difference is 
that every type in JavaScript can be initialized with var instead of a specific variable declaration as in 
Java.

In this example a number element will be logged in the console if the number is above 6 or below 0.
If it isn't, the number will be pushed (added) to the array discardedNumbers instead:
                            
var numbers  = [5, 3, 2, 7, 11, 12, 0, -20, 6];
var discardedNumbers = [];

for (i = 0; i < numbers.length; i++) {
    if (numbers[i] < 0 || numbers[i] > 6) {
        console.log(numbers[i]);
    } else {
        discardedNumbers.push(numbers[i]);
    }
}
console.log("Discarded numbers: " + discardedNumbers);

//Output: 
7
11
12
-20
Discarded numbers: 5,3,2,0,6

Example with modulo operator logging every second char in array - array, loop and if statement

In this example an array is given where every second char starting from b will be logged. That is done by 
iterating through the elements in the array with a for loop and then logging every second element using
the modulo operator:
                            
var letters = ["a","b","c", "d", "e", "f", "g", "h"];

for (i = 0; i < letters.length; i++) {
    if (i % 2 == 0)
        console.log(letters[i + 1]);
}

//Output:
b
d
f
h
